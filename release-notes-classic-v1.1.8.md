# UODreams Classic — Launcher & Client v1.1.8

Release del launcher self-contained e del client Classic per UODreams.

---

## Novità e correzioni rispetto a v1.1.7

### Launcher
- Il launcher **resta aperto** avviando qualsiasi assistant (**Orion**, **UOSteam**, **ClassicUO**, ClassicAssist, Razor Enhanced, ecc.)
- Fix crash **minimize/restore** (errore di paint alla ripristino finestra)
- Menu **Enhanced Map** — nuova voce **Richiedi Stanza** (pagina web integrata); voci del menu tradotte in italiano (**Avvia**, **Sfoglia…**, **Scarica**, **Richiedi Stanza**)

### Client
- **Rilevamento versione UO** — auto-detect in qualsiasi cartella client

### Parità UI da v1.1.7
- UI con **pill buttons** per gli assistant (stile outline)
- Pulsante **Buy me a coffee**
- Shard: **Modifica / +**, browse **...** sempre visibile, **cestino** per ogni path
- Pulsanti **Scarica** per assistant, centrati
- Cartella **`{Launcher}\Assistant\`**: download automatico al primo utilizzo
- Traduzioni **IT / EN**
- Launcher **self-contained**

---

## Come aggiornare

1. Apri il launcher e clicca **Aggiorna** (*Aggiorna* / *Update*)
2. Vengono aggiornati **launcher** e **client**
3. Impostazioni e **Profiles** restano **preservati** durante l'aggiornamento in-place

---

## Installazione (nuova installazione)

1. Scarica **UODreams-Classic-Launcher-v1.1.8.zip**
2. Estrai e avvia **UODreams Launcher.exe**
3. Al primo avvio il launcher scarica i componenti ClassicUO necessari
4. Scarica il client Ultima Online UODreams dal launcher (pulsante **Scarica** / **Download**)

### Server
- `login.uodreams.com:2593`

# UODreams Classic — Launcher & Client v1.1.8 (EN)

Release of the self-contained launcher and Classic client for UODreams.

---

## What's new since v1.1.7

### Launcher
- Launcher **stays open** when starting any assistant (**Orion**, **UOSteam**, **ClassicUO**, ClassicAssist, Razor Enhanced, etc.)
- **Minimize/restore** paint crash fix
- **Enhanced Map** menu — new **Request Room** item (embedded web page); menu items localized in Italian (**Avvia**, **Sfoglia…**, **Scarica**, **Richiedi Stanza**)

### Client
- **UO version detection** — auto-detect in any client folder

### UI parity from v1.1.7
- Refreshed UI with **outline pill buttons** for assistants
- **Buy me a coffee** button
- Shards: **Edit / +**, browse **...** always visible, **trash** icon per path
- Per-assistant **Download** buttons, centered
- **`{Launcher}\Assistant\`** folder: auto-download on first use

- **Self-contained** launcher

---

## How to update

1. Open the launcher and click **Aggiorna** / **Update**
2. Both **launcher** and **client** are updated
