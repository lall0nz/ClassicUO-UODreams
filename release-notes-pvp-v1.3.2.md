# UODreams PVP Launcher v1.3.2

## Novita' v1.3.2

### Desktop shortcut dopo rebrand 0nE UO
- Dopo OTA (e al prossimo avvio del launcher), il collegamento sul Desktop viene aggiornato automaticamente
- Icona impostata su `oneuo.ico` / icona 0nE UO (non piu' unicorna UODreams)
- Collegamento rinominato in **0nE UO Launcher**; rimossi duplicati legacy (`UODreams Launcher`, `UODreams Launcher pvp`, ecc.)
- Nessuna azione manuale richiesta: basta aggiornare a v1.3.2 via OTA e riavviare

### OTA / packaging
- Lo zip launcher include **entrambi** `0nE UO Launcher.exe` e `UODreams Launcher.exe` (stesso binario) per compatibilita' aggiornamenti da installazioni pre-1.3.1
- Profilo **Default PVP**: copiato solo se mancante — **mai** sovrascritto se gia' presente
- Profilo stock `default` **mai** toccato da OTA

### Client
- Nessuna modifica client in questa release (resta v1.3.1)

## English

### Desktop shortcut after 0nE UO rebrand
- After OTA (and on next launcher start), the Desktop shortcut is refreshed automatically
- Icon set to `oneuo.ico` / 0nE UO branding (no longer the old UODreams unicorn)
- Shortcut renamed to **0nE UO Launcher**; legacy duplicates removed (`UODreams Launcher`, `UODreams Launcher pvp`, etc.)
- No manual steps required: update to v1.3.2 via OTA and restart

### OTA / packaging
- Launcher zip ships **both** `0nE UO Launcher.exe` and `UODreams Launcher.exe` (same binary) for pre-1.3.1 install compatibility
- **Default PVP** profile: copy-if-missing only — never overwritten if already present
- Stock `default` profile never touched by OTA

### Client
- No client changes in this release (remains v1.3.1)
